# Нагрузочное тестирование с нуля

## Дисклеймер
Этот тайник знаний не претендует на абсолютную истину. Знания, описанные
здесь, собраны по крупицам в результате моего личного опыта, связанного
с нагрузочным тестированием в разных крупных компаниях. И каким-то
образом систематизированы, чтобы с ними можно было работать, базово
обучать людей и делиться опытом.
